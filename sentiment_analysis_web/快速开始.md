# 快速开始指南

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 初始化数据库

```bash
python init_db.py
```

这将从 `data/reviews.csv` 导入评论数据到SQLite数据库 `reviews.db`。

## 3. 运行应用

```bash
python app.py
```

## 4. 访问系统

打开浏览器访问: http://localhost:5000

## 功能页面

- **首页**: http://localhost:5000/
- **评论列表**: http://localhost:5000/reviews
- **情感统计**: http://localhost:5000/sentiment
- **词云分析**: http://localhost:5000/wordcloud
- **关于页面**: http://localhost:5000/about

## 注意事项

1. 确保已安装所有依赖包
2. 首次运行前必须执行 `init_db.py` 初始化数据库
3. 如果数据文件不存在，请先运行 `copy_data.py` 复制数据文件
