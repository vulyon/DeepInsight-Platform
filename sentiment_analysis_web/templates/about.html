关于系统

电商产品评论情感分析系统

系统介绍

本系统是一个基于Python Flask框架开发的电商产品评论情感分析平台，旨在帮助商家和用户快速了解产品评论的情感倾向，为产品改进和购买决策提供数据支持。

主要功能：

  • 评论数据管理和展示
  • 情感倾向分析（正面/负面）
  • 情感分布统计可视化
  • 词云分析展示

技术栈

后端技术：

Python Flask SQLite

前端技术：

Bootstrap ECharts jQuery

数据分析：

Pandas Jieba WordCloud Scikit-learn

功能特点

  • 支持大量评论数据的存储和查询
  • 实时情感分析统计
  • 直观的数据可视化展示
  • 响应式设计，支持多设备访问
